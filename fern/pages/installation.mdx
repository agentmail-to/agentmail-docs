## Quick Install

Install AgentMail using your preferred package manager:

```bash
# Using pip
pip install agentmail

# Using poetry
poetry add agentmail
```

## Prerequisites

- Python 3.8 or higher
- An AgentMail API key ([Get your API key](https://dashboard.agentmail.dev))

## Basic Setup

1. Install the package:

```bash
pip install agentmail
```

2. Set up your environment:

```python
from agentmail import AgentMail

# Initialize the client
client = AgentMail(api_key="your_api_key_here")

# Create your first inbox
inbox = client.create_inbox()
print(f"Your inbox address: {inbox.address}")
```

## Installation Options

### Using pip with requirements.txt

```bash
# Add to requirements.txt
agentmail>=1.0.0

# Install
pip install -r requirements.txt
```

### Using poetry

```bash
# Add dependency
poetry add agentmail

# Install
poetry install
```

### Development Installation

For contributing or development:

```bash
git clone https://github.com/agentmail/agentmail-python.git
cd agentmail-python
pip install -e ".[dev]"
```

## Verify Installation

Test your installation:

```python
from agentmail import AgentMail

# Should print the version number
print(AgentMail.__version__)
```

## Common Issues

- **API Key Not Found**: Make sure your API key is properly set in your environment variables
- **Import Errors**: Ensure you're using Python 3.8+
- **Connection Issues**: Check your internet connection and firewall settings
