AgentMail runs one hosted MCP implementation at `https://mcp.agentmail.to/mcp`. Smithery is an optional discovery and gateway surface for that same server, not a separate AgentMail MCP product or deployment.

See [AgentMail MCP](/integrations/mcp) for the canonical setup.
