## Quick Install

Install AgentMail using your preferred package manager:

<CodeBlocks>
```bash title="Python"
pip install agentmail
```

```bash title="Node"
npm install agentmail
```

</CodeBlocks>

## Prerequisites

- Python 3.8 or higher
- An AgentMail API key ([Request your API key](https://form.typeform.com/to/m0kKD0jH))

## Verify Installation

Test your installation:

```python
from agentmail import AgentMail

# Should print the version number
print(AgentMail.__version__)
```

## Basic Usage

Here's how to initialize the client and create your first inbox:

<CodeBlocks>
```python title="Python"
from agentmail import AgentMail

# Initialize the client

client = AgentMail(api_key="YOUR_API_KEY")

# Create an inbox

inbox = client.inboxes.create(domain="yourdomain.com")

````

```typescript title="TypeScript/Node.js"
import { AgentMailClient } from "agentmail";

// Initialize the client
const client = new AgentMailClient({ apiKey: "YOUR_API_KEY" });

// Create an inbox
await client.inboxes.create({
  domain: "yourdomain.com",
});
```

</CodeBlocks>

## Error Handling

Handle API errors gracefully with built-in error types:

<CodeBlocks>
```python title="Python"
from agentmail.core.api_error import ApiError

try:
    client.inboxes.create(domain="yourdomain.com")
except ApiError as e:
    print(f"Error {e.status_code}: {e.body}")

```

```typescript title="TypeScript/Node.js"
import { AgentMailError } from "agentmail";

try {
  await client.inboxes.create({ domain: "yourdomain.com" });
} catch (err) {
  if (err instanceof AgentMailError) {
    console.log(`Error ${err.statusCode}: ${err.message}, Message: ${err.body}`);
  }
}
```

</CodeBlocks>

## Async Support

Python users can also use our async client for non-blocking operations:

```python title="Python Async"
import asyncio
from agentmail import AsyncAgentMail

async_client = AsyncAgentMail(api_key="YOUR_API_KEY")

async def main():
    await async_client.inboxes.create(domain="yourdomain.com")

asyncio.run(main())
```
