# Frequently Asked Questions

<AccordionGroup>
  <Accordion title="How does email privacy work?">
    Each AI agent gets its own dedicated email address with built-in privacy controls. Emails are encrypted and securely stored, and you have full control over data retention and access policies.
  </Accordion>

{" "}

<Accordion title="Can I integrate with existing AI frameworks?">
  Yes! We offer native integration with LangChain and provide SDKs for popular
  programming languages. Our REST API allows for easy integration with any AI
  system.
</Accordion>

{" "}

<Accordion title="How does action detection work?">
  Our system automatically analyzes incoming emails using advanced NLP to
  identify actionable items, such as verification links, requested responses, or
  required attachments. These are returned as structured data via our API.
</Accordion>

{" "}

<Accordion title="What types of attachments can be processed?">
  We support parsing of common file formats including PDF, DOCX, XLSX, CSV, and
  images. Our smart parsing system extracts relevant information and presents it
  in a structured format for your agents.
</Accordion>

{" "}

<Accordion title="Is there a limit to the number of email addresses?">
  No, you can create as many email addresses as needed. Pricing is based on
  usage (number of emails processed and actions taken) rather than the number of
  addresses.
</Accordion>

  <Accordion title="Can agents send emails?">
    Yes, agents can send emails, but only in reply to emails they have received. They cannot send emails without receiving them first.
  </Accordion>
</AccordionGroup>
